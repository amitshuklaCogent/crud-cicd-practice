# CRUD Application with CI/CD Pipeline

This project is a web-based CRUD (Create, Read, Update, Delete) application developed using Laravel and Bootstrap 5. The application provides a user-friendly interface for managing records efficiently while following modern web development practices.

The project demonstrates the implementation of a complete CI/CD (Continuous Integration and Continuous Deployment) pipeline to automate code integration, testing, and deployment processes. Laravel is used as the backend framework to handle business logic, routing, and database operations, while Bootstrap 5 is used to create a responsive and modern user interface.

## Key Features

* Create, Read, Update, and Delete operations
* Responsive UI using Bootstrap 5
* Laravel MVC architecture
* Database integration with MySQL
* Form validation and error handling
* Automated CI/CD workflow
* Version control using Git and GitHub
* Secure and maintainable code structure

## Technologies Used

* Laravel
* PHP
* Bootstrap 5
* MySQL
* Git & GitHub
* CI/CD Tools (GitHub Actions/Jenkins)

## Objective

The primary objective of this project is to learn and implement CRUD operations in Laravel while gaining practical experience with CI/CD practices for automated software delivery and deployment.
